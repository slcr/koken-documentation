albums in sets don't display?

Theo Jan 27, 2014 08:19AM EST

I'm designing a custom .lens template which calls for albums with a certain category. However, when these albums are collected in sets, they don't load.

When I use the filter:category method they do, but the lightbox doens't work because of the custom template/url.

Is this incorrect use or is there a workaround?

2 Community Answers

Answered
Todd Dominey Jan 30, 2014 07:08PM EST

Hi Theo – I think I may have a solution for this. What you’d want to do is request the “albums” data source then add a “flat” filter so that albums inside of sets are returned as a single list. So you’d want to do something like this:

koken:load source=“albums” filter:flat=“true”
/koken:load

Inside that you’d use koken:loop to loop over all the returned albums, which should come back without their parent sets.
