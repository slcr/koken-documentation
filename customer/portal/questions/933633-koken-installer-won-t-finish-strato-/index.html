Koken installer won't finish (Strato)

Johannes Apr 09, 2013 05:11AM EDT

Hi,

I already wrote an email to the support team a couple of days ago and I haven't received any answer yet. You'll find any further informations needed in this email.

The installation process of the Koken does not finish. I followed guidelines and the troubleshooting tips and tried it about four times with different browsers and settings.

My provider Strato has a memory limit of 104 MB and a upload limit of 10MB (PHP 5.2.x; see http://www.strato-faq.co.uk/artikel.html?articleid=1569). I set a php.ini with the following content:
- - -
upload_max_filesize = 50M
post_max_size = 55M
safe_mode = Off
- - -

I just tried it once more with this settings and Firefox v15 in SafeMode. Five minutes later: nothing happens any more, same result.

Best,
Johannes

Brad Daily Apr 09, 2013 08:49AM EDT Koken developer

Strato by default uses PHP 5.2, and Koken seems to have issues with that version on Strato. Try using PHP 5.3 instead by adding this to the top of the .htaccess file:

AddType application/x-httpd-php53. php

Johannes,
I have had the same problem. It solved by PHP 5.3
